# MoneyQuest: Future Tycoon

A gamified financial education tool designed for kids and beginners.

## Features

- **The House**: Mortgage calculator with interactive charts.
- **Turbo Pay**: Debt payoff strategy simulator.
- **Money Tree**: Compound interest investment visualizer.

## How to Run

1. Simply open `index.html` in any modern web browser.
2. No installation or build steps required.

## Tech Stack

- React & ReactDOM (CDN)
- Tailwind CSS (CDN)
- Recharts (CDN)
- Lucide Icons (CDN)
